iterate

inline fun iterate(reversed: Boolean = false, apply: (T, PooledListIterator<T>) -> Unit): Unit

Parameters

reversed - if true, the iteration will begin from the last element and end with the first.

apply - will be applied to each element in the list. The action gets a direct reference to the list's PooledListIterator instance, which can be used to modify the list.
