insertAfter

fun insertAfter(element: T): Unit

Adds element after the value currently selected by the iterator. This utility method can be used ONLY during iteration with one of the iterator-caching methods: iterator or reversedIterator. Note that iterator is internally used by for-each loops and utilities like forEach method.

Parameters

element - will be added to the list after the current iteration element. If the iterator is not reversed, this will be the next element in the iteration.
