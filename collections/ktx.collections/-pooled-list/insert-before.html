insertBefore

fun insertBefore(element: T): Unit

Adds element before the value currently selected by the iterator. This utility method can be used ONLY during iteration with one of the iterator-caching methods: iterator or reversedIterator. Note that iterator is internally used by for-each loops and utilities like forEach method.

Parameters

element - will be added to the list before the current iteration element. If the iterator is reversed, this will be the next element in the iteration.
