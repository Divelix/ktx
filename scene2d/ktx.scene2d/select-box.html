selectBox

inline fun <I, S> KWidget<S>.selectBox(style: String = defaultStyle, skin: Skin = Scene2DSkin.defaultSkin, init: KSelectBox<I>.(S) -> Unit = {}): KSelectBox<I>

Parameters

style - name of the widget style. Defaults to defaultStyle.

skin - Skin instance that contains the widget style. Defaults to Scene2DSkin.defaultSkin.

init - will be invoked with the widget as "this". Consumes actor container (usually a Cell or Node) that contains the widget. Might consume the actor itself if this group does not keep actors in dedicated containers. Inlined. Allows to fill list's items.

I - type of items stored by this widget. Usually items are converted to string and displayed.

S - type of actor containers used by the parent. Usually Cell, Node or Actor.

Return
a SelectBox instance added to this group.
